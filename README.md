MVC Application in NodeJS
Software Setup:
1.Go to https://nodejs.org and download Node.js
installer and install it.
Node.js is a free, open-source, cross-platform
JavaScript runtime environment that lets
developers create servers, web apps, command line
tools and scripts.
2.Open command prompt (Run->cmd), create a
folder for project (md command), change directory
to project folder (cd command) and execute
following commands in it:
a. npm init -y
b.npm install express
c. npm install mysql2
d.npm install dotenv
e. npm install ejs
Note: npm is a package manager for Node.js packages, or
module.
3. Open Project folder in Visual Studio Code Editor or type
“code .” from project folder’s command prompt.

------------------------------------------------------------------
Note: Run MySQL Server or XAMPP MySQL service, create database (“mydb”), create table
users (id int, name varchar(30)).
To run the project execute “node app” from VS Code Editor’s terminal.
